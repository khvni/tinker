## Decisions Made

- Daily sweep use renderer timer + SQLite `memory_runs` on this branch. Reason: feature 04 scheduler not on `origin/main` base. Extraction + merge path isolated so scheduler can call same funcs later.
- Managed note writes go under `## Tinker Memory`. Reason: append-only, low collision with user sections, easy to parse/dedupe.
- Prompt injection use FTS search + one-hop `relationships` traversal from wikilinks. Reason: better context recall without embeddings.
- Session memory capture runs per completed chat turn, not only explicit session close. Reason: chat UI has long-lived sessions; this keeps vault memory landing without extra UX.
