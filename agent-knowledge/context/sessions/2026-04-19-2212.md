# Session ‚Äî 2026-04-19 22:12

## Goal
Refactor the entire `apps/desktop` UI to consume `@tinker/design` strictly. No ad-hoc palettes, no hand-rolled button/tab/toggle/input primitives. Preserve all feature behavior.

## Starting state

- `packages/design` already shipped 9 primitives (Button, IconButton, Badge, ClickableBadge, StatusDot, SegmentedControl, Toggle, TextInput, SearchInput) + tokens CSS (warm amber dark, Host Grotesk, 4px base).
- `apps/desktop/src/renderer/styles.css` carried a conflicting `--tinker-*` cyan-blue palette, radial cyan gradient hero bg, bespoke `.tinker-button*` classes with transparent gradient treatments, and Space Grotesk as the font.
- Every pane (`Chat`, `Settings`, `Today`, `FirstRun`, `VaultBrowser`, `Dojo`, `SchedulerPane`) + renderers (`MarkdownRenderer`, `MarkdownEditor`, `CsvRenderer`, `CodeRenderer`) used the hand-rolled classes and raw `<button>` / `<input>` elements.
- `Dojo` had its own hand-rolled `.tinker-dojo-tab` strip and a raw `<input type="checkbox">` for "active" switch.

## Changes landed

### Design tokens wired in
- `apps/desktop/src/renderer/styles.css` rewritten. First line now `@import "@tinker/design/styles/tokens.css";`. Legacy `--tinker-*` variables deleted.
- Structural `.tinker-*` classes kept (they encode layout) but every color / spacing / radius / font declaration now reads from `--color-*` / `--space-*` / `--radius-*` / `--font-*` tokens.
- Removed: radial cyan gradient background, `.tinker-pane` linear-gradient treatment, glass-blur modal backdrop, Space Grotesk stack, hand-coded rgba hex values.

### Primitive swaps (by file)
- `workspace/Workspace.tsx` ‚Äî header buttons ‚Üí `<Button variant=secondary size=s>`; header-meta pills ‚Üí `<Badge>`.
- `panes/FirstRun.tsx` ‚Äî all auth/vault CTAs ‚Üí `<Button>`.
- `panes/Settings.tsx` ‚Äî connect/disconnect buttons ‚Üí `<Button>`; auto-open preference toggle ‚Üí `<Toggle>`.
- `panes/Today.tsx` ‚Äî sweep button ‚Üí `<Button>`; vault status pill ‚Üí `<Badge>`.
- `panes/Chat.tsx` ‚Äî new-chat + save-as-skill + send ‚Üí `<Button>`; status chip ‚Üí `<Badge>`.
- `panes/VaultBrowser.tsx` ‚Äî "N files" pill ‚Üí `<Badge>`.
- `panes/Dojo.tsx` ‚Äî tab strip ‚Üí `<SegmentedControl>` (3 modes: browse / author / git sync, `preview` folded into `browse`); skill active switch ‚Üí `<Toggle>`; search ‚Üí `<SearchInput>`; all form fields ‚Üí `<TextInput>`; active-pill ‚Üí `<Badge>`; every button ‚Üí `<Button>`.
- `panes/SchedulerPane.tsx` ‚Äî create/edit/run/disable/delete + schedule hints ‚Üí `<Button>`; all form inputs ‚Üí `<TextInput>`; pill ‚Üí `<Badge>`.
- `renderers/MarkdownRenderer.tsx` ‚Äî Edit button ‚Üí `<Button>`.
- `renderers/MarkdownEditor.tsx` ‚Äî save-state pill ‚Üí `<Badge>` (color by dirty state).
- `renderers/CsvRenderer.tsx` ‚Äî pagination ‚Üí `<Button>`; row-count pill ‚Üí `<Badge>`.
- `renderers/CodeRenderer.tsx` ‚Äî language pill ‚Üí `<Badge>`.
