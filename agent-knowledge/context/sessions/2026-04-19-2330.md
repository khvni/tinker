# Session — 2026-04-19 23:30 — `@tinker/panes` scaffold + architectural pivot

## What happened

Evaluated three reference codebases and used the synthesis to refactor Tinker's architecture direction:

1. `anomalyco/opencode` (Tauri v2 + Solid.js) — portable patterns: Platform interface, ConnectionGate, session history windowing, sidecar `Channel<InitStep>` handshake. Non-portable: Solid JSX + Kobalte primitives.
2. `manaflow-ai/cmux` (Swift/AppKit + Bonsplit) — portable UX: vertical workspace sidebar, recursive binary split tree, `WorkspaceAttentionCoordinator` (rings / flashes / unread), structured sidebar metadata, AI-as-pane pattern. Non-portable: AppKit/NSView.
3. `superset-sh/superset` (Electron + React + zustand, ELv2) — portable patterns: `LayoutNode` tree + `PaneRegistry` type model, HOST/DEVICE split, coordinator pattern, `ask_user` overlay, co-located folder convention.
