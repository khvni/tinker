## Decisions

- Built as standalone primitive rather than extending `Badge`. Reasoning: `Badge` uses `--radius-md` (not pill), 8 unrelated variants, medium default size 32px. Adding a `shape` or `pill` axis would leak the gauge semantic into every `Badge` consumer. Three CSS rules vs. a shape-branching union — Elon step 2 favors keeping them separate.
- Tooltip uses native HTML `title` + full `aria-label`. No tooltip component needed for MVP.
- Tokens reused — did not need to add `--color-warning` variant because the spec explicitly calls for "brand accent" amber on the 50–80% band, which matches `--color-accent-*`.
- Numbers formatted with `toLocaleString()` and `font-variant-numeric: tabular-nums` so streaming updates do not jitter digits.
