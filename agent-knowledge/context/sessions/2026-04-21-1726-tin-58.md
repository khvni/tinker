# 2026-04-21 17:26 — TIN-58 M6.2 default memory root

## Scope

Deliver the MVP memory-root resolver for M6.2. The implementation must resolve the platform-specific default root, create it if missing, and expose the helper from `@tinker/memory`.

## Deliverable

- `packages/memory/src/memory-paths.ts` — resolves the M6.2 default root from the host data directory and creates it with the Tauri fs plugin.
- `packages/memory/src/memory-paths.test.ts` — covers Linux, macOS, Windows, and the create-if-missing behavior.
- `packages/memory/src/index.ts` — exports the new helpers.

## Implementation

- Used `process.platform` as the default selector, with a `dataDir()` fallback so the helper remains safe if `process` is unavailable at runtime.
- Added `ensureDefaultMemoryRoot()` so callers can resolve and create the directory in one step with recursive `mkdir`.
- Kept the change scoped to `@tinker/memory`; no renderer wiring added in this ticket.
