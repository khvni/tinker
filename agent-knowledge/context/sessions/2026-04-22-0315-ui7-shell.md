# Session — UI.7 workspace shell (TIN-182)

## Outcome

Workspace renders via a proper `<Titlebar>` primitive matching Paper 9I-0 (light) / 1-0 (dark) 1:1, instead of the bespoke `<header className="tinker-header">` + `<IntegrationsStrip compact />` boilerplate. Tauri window now uses `titleBarStyle: "Overlay"` + `hiddenTitle: true` so macOS traffic lights float over the custom titlebar. All M1 Dockview-retirement deps (TIN-8/14/9/10/11) landed upstream before this PR so the Workspace.tsx shell was finally safe to touch.

## What shipped

**New (folder-per-component under `apps/desktop/src/renderer/workspace/components/Titlebar/`):**

- `Titlebar.tsx` — 36px shell with three zones: 68px traffic-light spacer · center `Tinker` brand + monospace session-folder basename crumb · right cluster of 3 ghost IconButtons (New chat / Memory / Settings). `data-tauri-drag-region` on root; `data-tauri-drag-region="false"` on the actions cluster so IconButton clicks don't get swallowed by the drag region.
- `Titlebar.css` — tokens-only. bg = `--color-bg-panel`, border = `--color-border-subtle`, center text = `--font-size-base` (13) weight medium (500) line-height `--line-height-tight`. No raw hex/rgba. No magic numbers beyond 36/68 (Paper-match) with code comments.
- `Titlebar.test.tsx` — 9 tests: brand render, crumb on/off, trailing-separator trim, drag-region on root, drag-region-false on actions, three click handlers.
- `index.ts` — barrel.

**Modified:**

- `apps/desktop/src-tauri/tauri.conf.json` — added `"titleBarStyle": "Overlay"` + `"hiddenTitle": true`. macOS renders traffic lights over the custom titlebar; Windows/Linux fall through to default decorations.
- `apps/desktop/src/renderer/workspace/Workspace.tsx` — deleted the bespoke `<header className="tinker-header">` + `<IntegrationsStrip compact>` block. Mounts `<Titlebar>` at top of the workspace shell. Dropped the dead `sessions`/`mcpStatus`/`Badge`/`Button`/`IntegrationsStrip` destructures + imports.
- `apps/desktop/src/renderer/styles.css` — deleted `.tinker-header`, `.tinker-header h1`, `.tinker-header-meta`, `.tinker-workspace-integrations` + its media-query branch. Updated `.tinker-workspace-shell` grid to `auto 1fr` (was `auto auto 1fr`).
- `apps/desktop/src/renderer/routes/design-system.tsx` — new `titlebar` playground tab rendering three states (bare brand, with crumb, dark-theme wrapper).
