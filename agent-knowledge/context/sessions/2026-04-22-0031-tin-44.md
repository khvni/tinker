## Review loop

Reviewer (Elon 5-step + Buchan) flagged two real blockers + cleanups. All applied:

- **B1**: window shortcut hijacked typing inside inputs/textarea/contenteditable. Added `isEditableTarget()` guard + lowercased both sides of key comparison so future `Mod+K` specs match case-insensitively.
- **B3**: click-outside closed the panel on scrollbar-gutter clicks. Guarded with `event.target === documentElement|body && event.clientX >= documentElement.clientWidth` bypass.
- **B5**: playground had write-only `useState` for non-interactive pickers — removed.
- Simplifications: deleted mount-time `findInitialIndex` effect (clamp effect covers it); always return new state from `disableMouse` reducer branch (React bails via Object.is); inlined `flatten(groups)` → `groups.flatMap(g => g.items)`.
- CSS nit: `scroll-margin-top: 28px` → `var(--space-8)` for token discipline.
- Added 8th test covering "shortcut does not fire when focus is in an editable element".
