# TIN-152 — Session history windowing hook

## Outcome

Shipped a reusable React `useSessionHistoryWindow` hook and wired the current desktop `Chat` pane to use it with real OpenCode cursor pagination.

## What landed

- Added `apps/desktop/src/renderer/panes/useSessionHistoryWindow.ts`.
- Default window size is `100` messages with `25`-message upward reveals and `200px` top threshold.
- Hook exports cursor state `{ before, hasMore, isLoading, loadMore }`, preserves scroll position when revealing older content, and supports prepending older pages.
- `Chat.tsx` now fetches the latest 100 session messages, reads `x-next-cursor`, prepends older pages via `session.messages({ before })`, and renders through the bounded history window.
- Added `apps/desktop/src/renderer/panes/useSessionHistoryWindow.test.ts` covering window size, reveal math, append behavior, older-page behavior, and scroll preservation math.
- Updated `agent-knowledge/features/14-session-history-windowing.md` to match the shipped hook behavior and `agent-knowledge/context/tasks.md` row 14 to `review — TIN-152 · PR #51`.
