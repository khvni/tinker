# Session — TIN-83 + TIN-85

## What shipped

- Added renderer `useCurrentUser()` source of truth in `apps/desktop/src/renderer/useCurrentUser.ts`.
- App boot now blocks on current-user resolution and no longer keeps a second auth/session copy in `App.tsx`.
- Added `packages/memory/src/user-store.ts::listUsersByLastSeen()` + tests so auth restore can prefer the most-recent known user.
- Added native `restore_auth_session` Tauri command in `apps/desktop/src-tauri/src/commands/auth.rs`.
- Added Better Auth sidecar `POST /auth/refresh` in `packages/auth-sidecar/src/main.ts` for Google/Microsoft refresh-token validation.
- Workspace layout persistence now keys off the authenticated `currentUserId` instead of `local-user`.
